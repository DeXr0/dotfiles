# Installation
### Download and change folder
```
cd ~

git clone https://github.com/DeXr0/dotfiles.git

cd dotfiles
```
### One line install
```
cd ~ && git clone https://github.com/DeXr0/dotfiles.git && cd dotfiles && ./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
